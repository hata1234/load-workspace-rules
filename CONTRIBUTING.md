# Contributing

Thanks for helping improve Load Workspace Rules.

## Development Setup

This hook has no build step.

```bash
node --check handler.js
```

## Pull Requests

- Keep changes focused and explain which bootstrap behavior is affected.
- Preserve path-safety checks for absolute paths and traversal attempts.
- Avoid committing credentials, local user IDs, absolute local paths, or machine-specific configuration.
- Keep documentation public-facing and avoid embedding private operational policies.

## Security

Please do not open public issues containing secrets, tokens, private workspace contents, or sensitive logs. Report sensitive security findings privately to the repository owner.
